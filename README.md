# Poppo Live Streaming Platform

**Owner:** Kiran  
**GitHub:** [https://github.com/therealkiranda/poppo](https://github.com/therealkiranda/poppo)

## Description
Poppo is a live streaming platform where hosts can earn money and agents can earn commissions by recruiting hosts.  
The project includes:

- Host and agent signup
- Blog system
- Dynamic homepage editable via Filament admin panel
- Beautiful pink-themed homepage with video hero section
- Earnings section for hosts and agents
- Latest blogs display
